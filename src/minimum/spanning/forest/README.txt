Algorithm simply explained:

*We start with Prim's algorithm that is O(E*log(V))
*Loop through all the edges O(E)
*Loop through all the vertices O(V)
*Loop through all the trees O(V) if there are no edges.

Prim's algorithm is our most expensive method which is O(E*log(V))
